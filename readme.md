# Homeless

Hide annoyances on iOS
iOS 11 Only

## Features
- Hide Lockcreen HomeBar
- Hide Lockscreen Quick Actions
- Hide Swipe Up text
- Hide Clock and Date
- Hide Date only
- Hide In App HomeBar


## Installation 
Homeless is available on my beta repository  here: [mdaus.ch/repo](http://www.mdaus.ch/repo)

## Notes
Only tested on iPhone X 11.1.2
